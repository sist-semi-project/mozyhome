## 1. 개요
현대인들은 바쁜 일상 속에서도 집을 편안한 휴식처로 꾸미고 싶어합니다. 그에 따라 모던하고 편안한 분위기의 결합한 가구 쇼핑몰을 구현하게 되었습니다. 모지홈은 소비자가 원하는 제품을 쉽게 찾고 구매할 수 있는 서비스를 제공합니다.
